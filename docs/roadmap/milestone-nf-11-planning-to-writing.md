# Milestone NF-11 — Planning → Writing

*Status: **SPLIT** into [nf-writing-os/](nf-writing-os/00-overview.md) (2026-04-28)*

The original monolithic NF-11 grew to 22 stories spanning state contracts, claim ledgers, figure registries, academic categories, writing cockpits, and drafting companions. That's a programme, not a milestone. It has been split into a focused series under [`docs/roadmap/nf-writing-os/`](nf-writing-os/00-overview.md):

- **[NF-11 — Planning → writing handoff](nf-writing-os/nf-11-planning-to-writing.md)** — state contract, normalized writing plan, file-backed scaffold, chat-panel handoff. Closes the writer's immediate blocker.
- **[NF-12 — Claim / evidence ledger](nf-writing-os/nf-12-claim-evidence-ledger.md)** — tracked factual claims with source / status / verification.
- **[NF-13 — Figure & visual planning](nf-writing-os/nf-13-figure-planning.md)** — figure registry with spec ready for later image generation.
- **[NF-14 — Academic category](nf-writing-os/nf-14-academic.md)** — Textbook and Revision Guide book types.

The drafting companion / writing cockpit work folds back into the existing [Milestone 10](milestone-10-drafting-companion.md) rather than being reinvented. Items dropped from scope are tracked in [future-work.md](nf-writing-os/future-work.md).
